# Apple Royale

A 2.5D platformer built with Babylon.js and Havok physics. Jump between breaking branches as an apple.

## Run

```
npx serve . -l 3000
```

Open http://localhost:3000

## Demos

- 2.5D branch platformer (original): http://localhost:3000
- 3D rolling platformer (new): http://localhost:3000/?demo=3d
